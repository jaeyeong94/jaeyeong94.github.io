# jaeyeong94.github.io

Ted Ryu (유재영)의 온라인 이력서 · 포트폴리오.

- **URL**: https://jaeyeong94.github.io
- **Stack**: Next.js 15 · TypeScript · TailwindCSS 3 · pnpm
- **배포**: GitHub Actions → GitHub Pages (static export)
- **언어**: ko · en · zh · ja

## 로컬 개발

```bash
pnpm install
pnpm dev
```

## 빌드

```bash
pnpm build      # out/ 정적 파일 생성
pnpm typecheck  # TS 검증
pnpm lint       # ESLint
```

## 구조

자세한 구조와 설계는 [`plans/`](./plans) 참고.
